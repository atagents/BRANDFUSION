# Audio Assets

Suggested files for the current 3-video ambient pass:
- `raw/sea_noise.wav`
- `raw/construction_rumble.wav`
- `raw/welding_texture.wav`

Recommended format:
- WAV
- 48kHz
- stereo
- at least 20 to 60 seconds long
